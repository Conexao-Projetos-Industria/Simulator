## Commands to build on linux:

First install dependencies:
```
sudo apt install libglx-dev libgl1-mesa-dev libglfw3-dev libglew-dev cmake
```

Then build using cmake
```
mkdir build
cd build
cmake ..
cmake --build .
cd ..
```
